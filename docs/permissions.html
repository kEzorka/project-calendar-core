âš ï¸ NEVER-DELEGABLE Critical Permissions

The following permissions are marked as never-delegable due to their critical nature. These permissions cannot be delegated to other users under any circumstances and must be explicitly assigned by system administrators.

Implementation Code

Add this constant to your DelegationService or PermissionService:

const NEVER_DELEGABLE = [
  'project.manage_users.global',
  'task.delete.global',
  'user.manage.global',
  'permission.assign.global'
];

// Usage example
function canDelegate(permission) {
  return !NEVER_DELEGABLE.includes(permission.codeName);
}
